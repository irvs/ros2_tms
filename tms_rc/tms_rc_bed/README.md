# tms_rc_bed
## Attencion
This package is not using ROS2. (Because this scripts made for RaspberryPi zero).

このパッケージはRaspberryPi zero上で動かすため、ROS2を使っていません。

## SSH
```
ssh pi@192.168.4.130
```

## How to use (on RaspberryPi zero)

```
$ cd ~/ros2_ws
~/ros2_ws$ python3 ./src/ros2_tms/tms_rc/tms_rc_bed/scripts/tms_rc_bed.py
```
